# 🌐 Portfolio — Mahboob Alam

A modern, responsive personal portfolio website built to showcase my skills, projects, certifications, and contact information in a clean and interactive way.

---

## 🚀 Overview

This portfolio represents my work as a developer, focusing on clean UI, smooth user experience, and structured component-based architecture. It highlights my technical skills, projects, and achievements in a minimal yet professional layout.

---

## ✨ Features

- ⚡ Fully responsive design (mobile → desktop)
- 🌗 Dark / Light mode toggle
- 🧭 Floating navigation dock with smooth scrolling
- 🎯 Section-based layout (Hero, Skills, Education, Projects, Certifications, Contact)
- 📜 Expandable Certifications section (Show More / Show Less)
- 📧 Contact form with EmailJS integration (frontend email sending)
- 🎨 Modern UI with Tailwind CSS
- ✨ Subtle animations (twinkling background, hover effects)

---

## 🧱 Tech Stack

### Frontend
- React.js
- Tailwind CSS (v4 + Vite plugin)
- JavaScript (ES6+)

### Libraries & Tools
- React Icons
- EmailJS (for contact form)
- Vite (build tool)

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd <project-folder>
2. Install dependencies
npm install
3. Run the project
npm run dev

📧 EmailJS Setup (Contact Form)
Create an account on EmailJS
Create a service and template
Add environment variables:
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
Restart the project
